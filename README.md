# aimyon-archive

あいみょん 팬 아카이브 - 프로필, 디스코그래피, 곡별 스토리 아카이브

## 주요 서비스
- 아티스트 프로필 & 소개
- 앨범/곡 디스코그래피
- **Song Stories**: 곡별 가사번역 & 제작 스토리
- 최신 소식 & 뉴스

## 기술 스택
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- 향후 계획: JSP + Servlet → Spring Boot
